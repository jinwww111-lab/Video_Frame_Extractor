# 视频帧提取小工具

一个简单的 Python GUI 工具，用于从视频中提取、预览和选择帧。  

GUI 界面可选择本地视频文件，设置帧间隔并提取帧缩略图。  

缩略图可点击放大查看，左右切换及选中，选中所有需要保存的图片后，点击 **“保存选中图片”** 导出所有选中的帧到指定文件夹。  

（这是一个很简单的小工具，为了方便自己的实验数据处理做的，很粗略但好用）

---

一、下载项目

方式一：Git 克隆
```bash
git clone https://github.com/jinwww111-lab/video_frame_extractor.git
```

方式二：直接下载.py文件

二、运行程序

在项目目录下执行：
```bash
python video_frame_extractor.py
```

三、注意

1、请使用纯英文路径；

2、目前仅测试了MP4文件格式，不确定其他格式的可使用性。
